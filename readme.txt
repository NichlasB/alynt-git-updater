Enable automatic updates for WordPress plugins hosted on GitHub repositories.

== Description ==

Alynt Git Updater allows WordPress plugins hosted on GitHub to receive automatic updates through the WordPress dashboard, just like plugins from WordPress.org.

Features:
* Automatic plugin updates from GitHub repositories
* Support for private and public repositories
* Optional webhook support for immediate updates
* Easy setup with minimal configuration

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/alynt-git-updater` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure any GitHub-hosted plugins you want to update through the plugin settings

== Frequently Asked Questions ==

= Can this update plugins from private repositories? =

Yes, but you'll need to configure appropriate authentication.

= How do I set up a plugin for updates? =

Add the GitHub repository information to your plugin's header.
